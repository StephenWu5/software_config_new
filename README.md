### 说明

这是在工作中用到的软件的个性配置。


gvim-x64 是安装后的gvim8.0+ 内嵌一些插件

vim_ahk_master 是vim和autohotkey的配置

.tern-project

_vimrc 是vim的基本配置：插件，插件配置，基本设置和键盘映射；（重要）

autohotkey.ahk 是autohotkey的配置

vim_little_program 是vim或者说linux的命令行小程序【exe】

monokai.json 是vscode 的代码配置，来自黑马老师

settings-webstorm66.jar 是webstorm的配置，用的少；

火绒联网控制配置.json 如名字一样是联网控制；

markdown-to-richtext 是markdown转富文本的web在线工具

xmind8模版1  xmind8的模版文件  主要是解决默认字体的大小问题。

README.md 是目录说明
